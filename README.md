# Computational Intelligence (Python)

## ENG
Assignments for the Computational Intelligence course, written in Python.
These include, amongst other things:
- image recognition (classification, face recognition, analysis)
- creating simple machine learning models
- working with spreadsheets
- etc.

## PL
Zadania przedmiotu Inteligencji obliczeniowej, tworzone w języku Python.
Zawierają one między innymi:
- rozpoznawanie obrazów (ich klasyfikacja, rozpoznawanie twarzy, analiza)
- tworzenie prostych modeli uczenia masynowego
- praca na arkuszach kalkulacyjnych
- itp.
